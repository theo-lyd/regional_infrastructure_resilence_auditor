# Dashboard Question Bank and Batches

## Purpose

Link dashboard content to real decision questions from public-sector stakeholders.

## Question Bank

1. Which regions are currently most underserved across all service domains?
2. Which regions improved in childcare capacity over the last years?
3. Where is youth welfare capacity stagnant or declining?
4. Which specialties show highest hospital bed pressure?
5. Which regions have cross-sector imbalance (strong in one domain, weak in others)?
6. Which regions are forecasted to enter high-pressure risk bands?
7. Is current data freshness and completeness good enough for policy use?

## Dashboard Build Batches

### Batch D1: Executive Overview
- total maturity snapshot
- top underserved regions
- quick data quality status

### Batch D2: Childcare Capacity
- facility count trend
- approved places trend
- regional comparison and growth ranking

### Batch D3: Youth Welfare Coverage
- places by region and year
- coverage ratio and trend slope views

### Batch D4: Hospital Capacity
- beds by specialty and region
- concentration and growth trend analysis

### Batch D5: Cross-Sector Resilience
- service maturity index map/table
- underserved score ranking
- sector comparison matrix

### Batch D6: Predictive Risk
- forecasted pressure index
- near-term warning table by risk band

### Batch D7: Data Quality and SLA
- completeness and null-rate trends
- freshness compliance and breach log

## Acceptance Criteria per Batch

- every chart has documented source model
- every metric has dictionary entry
- every insight can be explained in plain language
