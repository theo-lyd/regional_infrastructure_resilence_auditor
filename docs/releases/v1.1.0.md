# Release v1.1.0

Release date: 2026-03-25

## Summary

This release introduces ingestion extensibility, schema-drift checks, SCD snapshot activation, expanded observability checks, domain quality scorecards, and documentation governance controls.

## Data Model Impacts

1. added `analytics_marts.mart_data_quality_scorecard_domain`
2. added dbt snapshots:
   - `snapshots.snap_dim_region`
   - `snapshots.snap_dim_sector`

## Ingestion Impacts

1. ingestion now reads from `data/reference/ingestion_source_registry.json`
2. supports both CSV and XLSX source adapters
3. schema drift checks can fail ingestion when contracts are violated

## Monitoring and Alerting Impacts

1. completeness regression observability check added
2. severity and dedupe/escalation model remains active

## Documentation Impacts

1. added release governance document
2. added shell and capability references to docs index
3. added docs sync CI guardrail script

## Known Limitations

1. ad-hoc query builder and user-defined metric builder remain not implemented
2. end-user what-if scenario UI remains not implemented
3. ingestion contract for new domains still requires staging/model extension
